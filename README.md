# Go counter app demo

## Backend

### Install

1. Download and install Go

- https://golang.org/

2. Set PATH

- https://go.dev/wiki/SettingGOPATH

3. Install dependencies

```bash
$ cd gc-backend
$ go get
```

### Run in dev mode
```bash
$ go run main.go serve
```

## Frontend
### Install

1. Install Node.js (18.13.0+)

2. Install dependencies

```bash
$ cd gc-frontend
$ npm run install
```

### Run in dev mode

```bash
$ npm run start
```
